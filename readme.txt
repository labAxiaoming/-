2个python文件：一个注册文件 与一个实时人脸识别文件
一个文件夹register 存储注册人信息   id+128维人脸特征向量


registertocsv.py文件运行后 输入名字拼音，然后会打开摄像头  人脸注册  
register文件夹下为注册信息

Videoreco.py是实时人脸识别脚本  本文件夹下 按住shift右键打开命令行输入python Videoreco.py  
                                           或者拉到anaconda的spyder中打开就可以运行
    鼠标点视频窗口，再按q可退出视频窗口
