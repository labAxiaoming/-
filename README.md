# 利用face_recognition库进行人脸注册及识别

#python3+ anaconda +opencv+ face_recognition

用法
运行 register.py 是利用摄像头来注册 注册得到的保存在know里面
运行 Videoreco.py 是识别的脚本 读取know/文件夹下图片 打开摄像头人脸识别与 /know文件夹下对比匹配
